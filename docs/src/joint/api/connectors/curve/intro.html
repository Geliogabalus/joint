The 'curve' connector interpolates route points using a catmull-rom curve converted into cubic bezier curve. Tangents at endings of a curve can be defined using several options.

Available options:

direction curve.Directions The overall direction of the curve. Defaults to curve.Directions.AUTO
sourceDirection curve.TangentDirections|g.Point|object|number The unit vector direction of the tangent at the source point. If the option is an object it must have x and y properties. If the option is a number, it represents an angle relative to the x-axis (counterclockwise).
targetDirection curve.TangentDirections|g.Point|object|number The unit vector direction of the tangent at the target point. If the option is an object it must have x and y properties. If the option is a number, it represents an angle relative to the x-axis (counterclockwise).
sourceTangent g.Point|object Vector of the tangent to the curve at the source point. Has priority over the sourceDirection and direction. If the option is an object it must have x and y properties.
targetTangent g.Point|object Vector of the tangent to the curve at the target point. Has priority over the targetDirection and direction. If the option is an object it must have x and y properties.
distanceCoefficient number Coefficient of the tangent vector length relative to the distance between points.

Example:

link.connector('curve', {
    direction: curve.Directions.HORIZONTAL // 'horizontal',
});
