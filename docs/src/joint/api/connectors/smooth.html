The 'smooth' connector interpolates route points using a cubic bezier curve. Without a route it uses options to render the link.

Available options:

direction string Direction of the connector's curve.

There is an exported Directions enum object that provides values for this option

  • Directions.AUTO - the default direction option. Changes orientation depending on the relative position of the source and target
  • Directions.HORIZONTAL - fixes the direction in the horizontal orientation
  • Directions.VERTICAL - fixes the direction in the vertical orientation
  • Directions.LEGACY - enables a legacy connection render algorithm
minOffset number Minimal curve point offset in case of source and target proximity. The default is 100.

Example:

link.connector('smooth', {
    minOffset: 80,
    direction: Directions.HORIZONTAL // 'horizontal'
});

(Deprecated) For the purposes of backwards compatibility, the 'smooth' connector may also be enabled by setting the link.smooth property to true.

// deprecated
link.set('smooth', true)
